Tips1.alfredworkflowファイルの作成方法
```
前提：
 Alfred.appのインストール
 powerpackライセンスの購入とアクティベート

作り方：
　Alfred4にはエキスポートの機能が実装されています
 
　1.サイドメニューのワークフローを選択
　2.右クリック（コンテキストメニュー）→ export...
  （もしくは右上のエキスポートアイコンをクリック）
```
